This project analyzes the **Centers for Medicare & Medicaid Services (CMS) Medicare Part B Physician/Supplier dataset for 2024 to understand healthcare service utilization and reimbursement patterns across provider specialties. The dataset contains information on HCPCS procedure codes, number of allowed services, allowed charges, and Medicare payment amounts submitted by physicians and suppliers under Medicare Part B.

The objective of this analysis is to explore how different specialties contribute to overall Medicare spending and to identify patterns in service volume and reimbursement levels. Using Excel, I created pivot tables, calculated fields, and visual dashboards to summarize metrics such as total allowed services, total allowed charges, and Medicare payments by specialty and procedure code. I also derived additional indicators, including average price per service, to better compare reimbursement levels across procedures.

In addition, SQL queries were used to perform deeper analysis on the dataset, including identifying high-volume but low-reimbursement procedures, ranking specialties based on average service price, and examining variation in payment structures across commonly billed HCPCS codes. The insights from this analysis help highlight how service utilization and reimbursement vary across medical specialties within the Medicare Part B program.

This project demonstrates practical skills in healthcare claims data analysis, data transformation, SQL querying, and data visualization, which are essential for healthcare data analyst and clinical data analyst roles.
